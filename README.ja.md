# ytdl-web

[English](README.md) | [简体中文](README.zh-CN.md) | 日本語

Flask ベースの yt-dlp Web ダウンローダーです。  
単一動画・プレイリスト、進捗追跡、プロキシ設定、多言語 UI、ブラウザ経由のローカル保存に対応しています。

## プロジェクト概要

- バックエンド: Flask + yt-dlp
- フロントエンド: HTML/CSS/JavaScript
- 設定保存: SQLite
- タスク状態管理: メモリ上のタスクマネージャー
- 既定ポート: `8000`

## 主な機能

- 動画/プレイリスト情報と形式一覧の取得
- 形式、音声トラック、字幕、高度なオプションの選択
- ダウンロード速度・残量・ETA を含む進捗表示
- タスクのキャンセルとログ確認
- 生成ファイルの単体/一括ダウンロード
- 多言語 UI（英語/中国語/日本語を含む）
- 任意のプロキシ設定（`none/http/socks5`）

## 動作要件

- Python 3.8+
- `pip`
- `ffmpeg` を PATH に設定推奨（結合/後処理に使用）

## インストール

```bash
pip install -r requirements.txt
```

## 起動方法

### 方法 A: 自動ランチャー（推奨）

Windows:

```bat
run.bat
```

- ローカルの Python を自動検出
- 番号でインタプリタを選択
- `app.py` を起動

番号指定で直接起動:

```bat
run.bat 1
```

Linux/macOS:

```bash
chmod +x run.sh
./run.sh
```

### 方法 B: 直接起動

```bash
python app.py
```

## ブラウザで開く

起動後:

- ホーム: `http://127.0.0.1:8000/`
- ヘルスチェック: `http://127.0.0.1:8000/api/health`

## 基本的な使い方

1. Web ページを開く
2. 設定で保存先・プロキシ・言語を設定
3. YouTube URL を貼り付けて情報取得
4. 形式とオプションを選択
5. ダウンロード開始、進捗を確認
6. 生成ファイルをローカルへ保存

## 補足

- 書き込み権限エラーが出る場合は、書き込み可能なフォルダを使用してください。
- 長いタイトルや特殊文字を含むファイル名向けに、Windows での安全なファイル名処理を適用しています。
